# 每週背單字確認表

這是一個可加入 iPad 主畫面的 PWA App。

## 使用方式

1. 用 Safari 打開 GitHub Pages 網址。
2. 點分享。
3. 選「加入主畫面」。
4. 從主畫面圖示開啟。
